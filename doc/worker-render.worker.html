A module that runs within your Web Worker to handle communication with the Window.

Object

Import this module and call the ready function to start your application. You can use any JavaScript framework to render your application as a virtual DOM is implemented that looks exactly like the real DOM.

Example

var workerRender = require("worker-render/worker");
var $ = require("jquery");
var app = require("./app");

workerRender.ready(function(){
    $("body").html(
        app()
    );
});
