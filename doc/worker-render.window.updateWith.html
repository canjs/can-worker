updateWith

  • function
worker-render.window.updateWith  
  • source

Connect a worker with the window's document. The function facilitates message passing between the window and the worker so that any events that occur within the window are passed to the worker and any changes to the DOM that happen in the worker are applied on the window side.

windowRender.updateWith(worker)

Parameters

  1. worker {Worker}

    A Web Worker where your application is running.
